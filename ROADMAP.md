# 🎯 开发路线图 (Roadmap)

> OpenClaw Pixel Office - 从原型到 v1.0

---

## Phase 1: MVP 原型 (已完成 ✅)

**目标：** 让项目能在 Godot 中运行起来

- ✅ 基础项目结构
- ✅ 主场景 + 相机控制
- ✅ 多层办公室环境生成
- ✅ 3 个 Agent 角色（方块人）自由移动
- ✅ 楼层切换 (1-5 键)
- ✅ 时间系统 + 加速 (空格键)
- ✅ 任务板基础读取（模拟+真实文件）
- ✅ 首次 commit + GitHub 推送

**里程碑：** 2026-03-09 ✅

---

## Phase 2: 核心玩法完善 (预计 2-3 周)

### Sprint 2.1: Agent 行为升级
- [ ] 为每个 Agent 添加独特的颜色和表情（用文字/简单图形）
- [ ] 任务状态 → 动作映射精准化
  - `IN_PROGRESS` → 持续打字动画
  - `BLOCKED` → 踱步 + 红色闪烁
  - `DONE` → 庆祝动作（跳一下）
- [ ] 思想气泡显示具体任务标题
- [ ] Agent 之间的"社交"（偶尔去别人工位聊天）

### Sprint 2.2: UI 与交互
- [ ] 完善 UI 层：显示当前时间、任务总数、各状态计数
- [ ] 添加鼠标悬停 Agent 时的详细信息面板
- [ ] 点击 Agent → 弹出控制菜单（查看任务、强制休息、重启）
- [ ] 调试控制台实现（`~` 键打开）
- [ ] 添加 Godot 原生设置对话框（修改配置）

### Sprint 2.3: 任务板深度集成
- [ ] 正确解析 `TASK_BOARD.md` 表格（支持更多格式）
- [ ] Task 更新时，Agent 状态实时变化（≤2秒延迟）
- [ ] 任务完成时，Agent 头顶显示 🎉 特效
- [ ] 在游戏中可视化任务板（墙上贴的表格贴图）
- [ ] 添加任务过滤（只显示玩家的任务 / agent 的任务）

**Phase 2 完成标志：**
- 所有 Agent 行为反映真实 OpenClaw 状态
- 玩家可以通过鼠标与游戏交互
- 任务流转动画流畅

---

## Phase 3: 美术与音效 (预计 1-2 个月)

### 3.1 像素美术外包/自制
**优先级：**
1. **Agent 角色**（16x32 或 24x36 像素）
   - Jarvis：企鹅/机器人造型，穿西装
   - Corali：女秘书，戴眼镜，拿文件夹
   - Xiaoyuan：极客青年，大耳机，多屏幕
   - Healthcheck：白大褂医生，拿听诊器
   - Github：Octocat 猫

2. **办公室 tileset**（砖块/地砖/家具）
   - 地板纹理（木地板 / 地毯）
   - 办公桌（2种样式）
   - 椅子
   - 电脑显示器
   - 窗户、绿植

3. **UI 元素**
   - 像素风字体（点阵）
   - 任务卡片边框
   - 按钮、面板装饰

**资产规格：**
- 角色：48x72 px（3倍放大，适合高清）
- 场景：1280x720 像素完美
- 格式：PNG（带透明度）

### 3.2 音效与音乐
- [ ] 键盘敲击声（loop）
- [ ] 消息通知声
- [ ] 咖啡机冲泡声
- [ ] 任务完成音效（success chime）
- [ ] 背景音乐：
  - Day：轻快 8-bit（类似动物森友会）
  - Night：舒缓 lo-fi
- [ ] 错误警报音

**资源来源：**
- [OpenGameArt.org](https://opengameart.org) (CC0 免费)
- [Chip32](https://chip32.org) (chiptune generator)
- [bfxr](https://www.bfxr.net/) (音效生成)

## Phase 4: 功能扩展 (2-3 个月)

### 4.1 经济与升级系统
- [ ] 任务完成后获得"算力币"
- [ ] 商店系统：用硬币升级 agent 能力
  - Jarvis：更快的消息处理（动作加速）
  - Corali：自动脚本修复（错误率减少）
  - Xiaoyuan：AI 搜索助手（工作进度+30%）
- [ ] 装饰品购买（买盆栽、海报）

### 4.2 成就系统
- [ ] 成就面板（弹窗展示）
- [ ] 50+ 成就预设（见 README）
- [ ] 成就解锁动画
- [ ] 导出成就截图

### 4.3 多结局
- [ ] 根据 30 天运营评分（可用率、任务完成率、agent 满意度）
- [ ] 3 个结局：完美、平庸、崩溃
- [ ] 结局动画（烟花 / 破产清算 / 庆功宴）

### 4.4 实时同步 (WebSocket)
- [ ] OpenClaw 启动 WS 服务器（port 8080）
- [ ] 游戏连接并订阅 `task_updated`、`agent_activity` 事件
- [ ] 零延迟状态同步（替代文件轮询）
- [ ] 断线重连 UI 提示
